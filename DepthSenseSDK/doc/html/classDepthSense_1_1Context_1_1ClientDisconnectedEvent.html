Detailed Description

The clientDisconnected event is raised when a client disconnects from the DepthSense server.

Parameters:
appNamethe name of the client executable, or an empty string if it could not be determined
pidthe process ID of the client, or -1 if it could not be determined
sourceIPthe source IP address
sourcePortthe source IP port
See also:
ClientConnectedEvent
